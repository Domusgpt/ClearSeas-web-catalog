1. TESSERACT (8-cell)
   - 4D hypercube with 16 vertices
   - 32 edges connecting all vertices
   - Perfect 4D symmetry
   - Assigned to: .signal-card, .step

2. 24-CELL
   - Unique 4D polytope (no 3D analogue)
   - 24 vertices, 96 edges
   - Exceptional symmetry properties
   - Assigned to: .capability-card

3. 5-CELL (4D Simplex)
   - Simplest regular 4D polytope
   - 5 tetrahedral cells
   - 10 edges total
   - Assigned to: .platform-card

4. 120-CELL
   - Most complex regular 4D polytope
   - 120 dodecahedral cells
   - Simplified subset for performance
   - Assigned to: .research-lab

MATHEMATICAL FOUNDATION
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

4D Rotation Matrices:
  Matrix4D.rotateXW(θ)  - Rotation in XW plane
  Matrix4D.rotateYW(θ)  - Rotation in YW plane
  Matrix4D.rotateZW(θ)  - Rotation in ZW plane
  Matrix4D.rotateXY(θ)  - Rotation in XY plane

Stereographic Projection (4D → 3D):
  x' = x * d / (d - w)
  y' = y * d / (d - w)
  z' = z * d / (d - w)
  where d = 2.0 (distance parameter)

Perspective Projection (3D → 2D):
  Standard perspective matrix with FOV = π/4


SHADER PIPELINE
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

VERTEX SHADER:
  1. Apply 4D rotation matrices to vertices
  2. Project from 4D to 3D (stereographic)
  3. Apply 3D rotation matrices
  4. Apply perspective projection
  5. Calculate depth for fragment shader
  6. Set point size based on depth

FRAGMENT SHADER (Points):
  1. Render circular points (discard outside radius)
  2. Interpolate colors based on depth
  3. Apply temporal shimmer effect
  4. Soft edge falloff for anti-aliasing
  5. Depth-based alpha for transparency

FRAGMENT SHADER (Lines):
  1. Interpolate colors based on depth
  2. Depth-based alpha for edge fade
  3. Maintain holographic aesthetic
