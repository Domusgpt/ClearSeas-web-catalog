## ‚ö° Pro Tips

1. **Always use SVG logos** when possible (infinite scaling, tiny file size)
2. **Navy must dominate** every composition (70%+ background)
3. **Aqua is an accent** (10-20% maximum)
4. **Test at small sizes** (icons should work at 16px)
5. **Interactive art** can be static screenshot or live iframe
6. **Patterns tile seamlessly** (CSS `background-repeat: repeat`)
